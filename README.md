# greetings
Detects a known face and greets.

## Requirements

* Python 3.7
* OpenCV 3.4
* mediapipe 0.8

## Quick Start

1. Clone repo

```sh
git clone https://github.com/briandiaz/greetings.git
cd greetings
```

2. Create a virtual environment

```sh
python3 -m venv .venv
source .venv/bin/activate
````

3. Install dependencies

```sh
python3 -m pip install -r requirements.txt
```
